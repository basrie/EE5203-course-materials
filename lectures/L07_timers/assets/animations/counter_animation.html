Notes for class
  • CNT increases toward ARR in up‑mode; on match it issues update and reloads.
  • Down‑mode mirrors this: when CNT hits 0, it updates and reloads from ARR.
  • This demo abstracts prescaler: the slider increases the tick period to simulate slower timer clocks.
